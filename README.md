# Weather App

Weather App is a simple web application that retrieves and displays weather information using a public API.

The project focuses on understanding API requests, DOM manipulation and clean front-end structure.

## Features

- Search weather by city
- API integration
- Dynamic data rendering
- Responsive layout
- Error handling for invalid input

## Tech Stack

- HTML
- CSS
- JavaScript
- External Weather API

## Installation

1. Clone the repository
2. Open index.html in your browser

No build tools required.

## What I learned

- Working with external APIs
- Fetch requests in JavaScript
- Handling asynchronous code
- Structuring small front-end projects
- Basic error handling
